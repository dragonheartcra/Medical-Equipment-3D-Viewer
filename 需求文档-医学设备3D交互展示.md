## 文档关系与阅读顺序

当前项目有 3 个主要文档：

1. `需求文档-医学设备3D交互展示.md`：需求源头，记录 5 台设备、部件、图片层级和交互要求。
2. `项目交接文档-完整版.md`：早期完整交接资料，包含素材整理、参考项目、原始技术设想等历史信息；其中 React/R3F、压缩 GLB 等部分已不再是当前实现方案。
3. `当前开发交接文档-医学设备3D展示.md`：当前最新接手入口，记录实际代码状态、已实现功能、调试模式、已确认标注点和后续建议。

后续接手时，优先阅读 `当前开发交接文档-医学设备3D展示.md`，需要核对原始需求时再回看本文档。

## 一、总览

5台设备：**DR、CT、DSA、MRI、超声**

整体交互模式：
- **悬浮**（hover）→ 显示部件外观 / 一级图
- **点击** → 显示内部结构 / 下一级图
- **再点** → 更深一层细节（如果有）

---

## 二、逐设备交互说明

### 1. DR（数字化X线摄影）

| 要素 | 内容 |
|------|------|
| 整体3D | 有一张DR整体立体图 |
| **焦点部件1** | **X线球管** |
| 层级 | ① 球管外观立体图 → ② 内部结构图 → ③ 更细内部结构图 |
| 触发 | 鼠标移到整体图的球管位置 → 显示①；点击 → 显示②；再点 → 显示③ |
| 文字 | 每层配不同说明文字 |
| **焦点部件2** | **平板探测器** |
| 层级 | ① 平板探测器外观 → ② 非晶态硅 → ③ 非晶态硒 |
| 触发 | 同上，悬浮→①，点→②，再点→③ |

### 2. CT（计算机断层扫描）

| 要素 | 内容 |
|------|------|
| 整体3D | 有一张CT整体立体图 |
| **焦点部件1** | **控制面板** |
| **焦点部件2** | **扫描机架** |
| 层级 | ① 机架透视图 → ② CT扫描图 |
| 触发 | 悬浮机架区域 → 显示透视图；点击 → 显示CT扫描图 |

### 3. DSA（数字减影血管造影）

| 要素 | 内容 |
|------|------|
| 整体3D | 一张DSA整体立体图 |
| 交互 | **每个标注出来的小部件**，点击后单独展示该部件的立体图 |

### 4. MRI（核磁共振成像）

| 要素 | 内容 |
|------|------|
| 整体3D | MRI整体大机器 |
| **主磁体线圈** | 点击整体机器 → 显示主磁体线圈 |
| **射频线圈** | 3类射频线圈，看能不能单列在旁边展示 |

### 5. 超声

| 要素 | 内容 |
|------|------|
| 整体3D | 超声整体 |
| **探头** | 探头外观 |
| **探头内部** | 探头内部结构 |

## 四、待生成的3D模型清单

需要生成「整体3D」的：DR整体、CT整体、DSA整体、MRI整体、超声整体

需要生成「部件3D/特写」的（可用平面图，不一定都要3D）：
- DR：X线球管①~③、平板探测器①~③
- CT：控制面板、扫描机架①②
- DSA：各标注部件（至少3~5个主要部件）
- MRI：主磁体线圈、射频线圈3类
- 超声：探头、探头内部

**总计**：约 20 张平面图 → 约 8~10 个3D模型

## 五、网页最低实现方案

### 交互设计（每个设备一个页面）

```
┌─────────────────────────────────────┐
│  [←返回]          MRI 核磁共振      │
├────────────────────┬────────────────┤
│                    │                │
│   3D 主模型        │  侧边/弹窗     │
│   (可旋转缩放)     │  详情展示区    │
│                    │                │
│   ● 标注点1       │  点击后显示    │
│   ● 标注点2       │  对应层级图片  │
│                    │  + 文字说明    │
├────────────────────┴────────────────┤
│  设备简介文字                       │
└─────────────────────────────────────┘
```

### DR / CT 交互流

```
悬浮标注点 → 侧边显示部件外观图 + 简介
    │
    点击 → 侧边切换为内部结构图 + 新文字
    │
    再点 → 侧边切换为更深层图 + 新文字
```

### DSA 交互流

```
点击标注点 → 弹窗/侧边显示该部件的独立大图
```

### MRI 交互流

```
点击整体模型 → 侧边显示主磁体线圈
射频线圈3类 → 侧边常驻/切换按钮
```

### 超声交互流

```
点击探头 → 显示探头内部结构
```

## 七、关键决策

| 决策点 | 建议 |
|--------|------|
| 混元3D vs Tripo？ | 先测混元（免费），效果不行再 Tripo（闲鱼会员 ~¥20/月） |
| 所有设备都做3D？ | 整体5台做3D，内部细节优先用现有图片，效果好再补3D |
| 标注点怎么做？ | 3D模型上放 Sprite 标签，悬停/点击控制侧边栏 |
| 压缩目标？ | 单个 GLB < 2MB |
